# To-Do List Application

This is a simple desktop-based To-Do List project developed using Python and Tkinter.

## Features
- Add tasks
- Delete tasks
- View tasks in a list
- Simple GUI interface

## Technologies Used
- Python
- Tkinter

## Author

## How to Run the Project

1. Install Python
2. Download the project files
3. Run the following command:

python todo list.py
